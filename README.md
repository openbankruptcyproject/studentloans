# studentloans.openbankruptcyproject.org

**Student Loan Discharge in Bankruptcy**

Consumer-facing educational hub covering § 523(a)(8) adversary proceedings, the Brunner test, federal vs private student loans, and the alternate federal discharge paths.

Live site: <https://studentloans.openbankruptcyproject.org/>

Maintained by the [Open Bankruptcy Project](https://openbankruptcyproject.org/), a 501(c)(3) public charity (EIN 41-5159631).
